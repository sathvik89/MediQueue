# MediQueue
A real-world hospital queue management system designed to digitize patient registration, appointment scheduling, and real-time queue tracking. The system is being developed using modern web technologies and follows core Object-Oriented Programming principles, all five SOLID principles, and key design patterns such as Observer (for live queue updates and notifications), Strategy (for flexible queue handling), and Factory (for scalable object creation). Patients can seamlessly book appointments, track their live queue position, and receive timely notifications, while doctors are provided with a dedicated dashboard to efficiently manage patient flow and workload.
